## Description
This project analyzes Python as an open-source software and demonstrates Linux shell scripting using 5 scripts.

## Scripts

1. Script 1 - Displays system information  
2. Script 2 - Checks Python installation  
3. Script 3 - Audits directories and permissions  
4. Script 4 - Analyzes log files  
5. Script 5 - Generates open-source manifesto  

## How to Run

Make scripts executable:
chmod +x *.sh  

Run scripts:
./script1.sh  
./script2.sh  
./script3.sh  
./script4.sh /var/log/Xorg.0.log error  
./script5.sh  

## Requirements
- Linux VM  
- Bash shell  
- Python installed
